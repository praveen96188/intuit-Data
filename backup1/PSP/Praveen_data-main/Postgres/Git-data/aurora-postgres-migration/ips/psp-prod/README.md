# Welcome to the IPS Datastore Client!

This is an AgI repo that was created for you.  It contains a CDK application that will deploy all your 
datastore needs. 

## Deploy something!

Please check the [IPS Quick Start Guide](https://github.intuit.com/data-dataplatform/ips-datastore-client/wiki/Quick-Start-Guide)
for next steps.

If you have any issues, feel free to reach out to [#team-ips-aurora](https://intuit-teams.slack.com/archives/C01K9KE4FD3)!

## Useful Commands

 * `mvn package`     compile and run tests
 * `cdk ls`          list all stacks in the app
 * `cdk synth`       emits the synthesized CloudFormation template
 * `cdk deploy`      deploy this stack to your default AWS account/region
 * `cdk diff`        compare deployed stack with current state
 * `cdk docs`        open CDK documentation

Enjoy!
