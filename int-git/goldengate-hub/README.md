# goldengate_hub

This repository is used to create Goldengate Hub in AWS. This is mainly for Oracle RDS/database.
The following diagram shows the most common use case of Goldengate Hub. For DR purpose, Goldengate Hub HA is highly recommended. Goldengate Hub standalone (non-HA) can be created also.
