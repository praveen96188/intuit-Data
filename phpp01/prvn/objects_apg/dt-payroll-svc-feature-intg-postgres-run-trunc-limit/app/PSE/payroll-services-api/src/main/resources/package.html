PSP API
This package contains the specification of the Payroll Services Platform API.

Concepts
The API is organized around three concepts – managers, data transfer objects (dtos) and finders.

Managers (see {@see com.intuit.sbd.payroll.psp.api.managers}) implement the service methods that can be used to modify, delete or add data to PSP.
DTOs (see {@see com.intuit.sbd.payroll.psp.api.dtos}) implement the parameters to the service methods PSP.
Finders (see {@see com.intuit.sbd.payroll.psp.api.finders}) implement the methods that can be used to query for PSP’s data.
