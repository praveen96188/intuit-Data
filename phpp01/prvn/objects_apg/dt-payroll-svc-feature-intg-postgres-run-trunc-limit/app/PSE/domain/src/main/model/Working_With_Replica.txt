1. Force sync PSP.eap
2. Remove read-only flag on PSP.eap
3. Remove read-only flag on PSP.bedl.xml
4. Open PSP.eap and go to Tools --> Manage .EAP file --> Synchronize Replica and choose the replica file
5. Generate PSP.bedl.xml
6. To make changes in the model, open PSP-replica.eap, make changes, and repeat steps 3-4.
7. To merge the model to the main line from dev, check-out the PSP.eap in main and synch the PSP-replica.eap to it.
